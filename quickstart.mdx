## Use TuringDB for the first time

<Steps>
    <Step title="Get your API credentials">
    Sign up at console.turingdb.ai, then go to API Keys to retrieve your secret API key.
    </Step>
    <Step title="Clone the Quickstart repo">
```shell
git clone https://github.com/turing-db/quickstart && cd quickstart
```
    </Step>
    <Step title="Run the project">
```shell
uv run
```
    </Step>
    <Step title="You're done!">
    </Step>
</Steps>
