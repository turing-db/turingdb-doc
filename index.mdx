## Use TuringDB with Claude Code

Install the official TuringDB Claude Code skill so Claude can drive TuringDB for you (start the server, query, write, run vector search):

```bash
npx skills add https://github.com/turing-db/turingdb-skills
```

Then invoke it in any Claude Code session with `/turingdb <your task>`. See the [Claude Code Skill](/claude-skill) page for details.
