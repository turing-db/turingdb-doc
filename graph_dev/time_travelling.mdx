---
title: "Time Travel"
sidebarTitle: "Time Travel"
description: "You can time travel in you graph databases and see the commit history"
---

<Tabs>
  <Tab title="Cypher">
    ```jsx
    // Show commit history
    CALL db.history()
    
    // Read a previous commit
    checkout c0a6ce3c0315be7a
    
    // Query graph at specific commit
    MATCH (n:Person {Name:'Jane'})-[e:KNOWS]->(m:Person {Name:'John'}) RETURN n, e, m")
    ```
  </Tab>
  <Tab title="Python SDK">
    ```python
    # Show commit history
    client.query("CALL db.history()")
    
    # Read a previous commit
    client.checkout(change="main", commit="c0a6ce3c0315be7a")
    
    # Query graph at specific commit
    client.query("MATCH (n:Person {Name:'Jane'})-[e:KNOWS]-(m:Person {Name:'John'}) RETURN n, e, m")
    ```
  </Tab>
</Tabs>
