<Tabs>
    <Tab title="Python SDK">
    ```python
    # Create graph object
    from turingdb import TuringDB
    
    # setup client
    client = TuringDB(
     instance_id="...", # Replace by your instance id
     auth_token="...",  # Replace by your API token
    )
    
    # Create a new graph
    client.query('CREATE GRAPH mygraph')
    client.set_graph('mygraph')
    ```
  </Tab>
